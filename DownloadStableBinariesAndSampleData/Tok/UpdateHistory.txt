2018.08.31. 처음 배포
Tok의 특징과 주요 기능은 아래와 같습니다.
0. 다수의 텍스트 파일을 처리할때 사용
1. GIS 관련 기능을 모두 제외. 그래서 ASCII 텍스트 파일, 일반 텍스트 파일을 대상으로 함
   (단, TIFF와 ASCII 파일의 변환기능은 포함 - GDAL 이용, 다른 S/W에 있는 기능은 가능한 제외..)
2. ascii 파일 변환 - TIFF<->ASCII, clipping, resampling
3. ascii 파일에서 정보 추출
    - 사용되는 ascii 파일들은 모두 같은 해상도와 영역을 대상으로 해야 합니다.
    - 특정 위치의 셀 값, 특정 영역의 평균값
    - 전체 ascii 파일 누적,  ascii 파일 몇 개씩 누적
4. 텍스트 파일에서 어떤 조건에 위치한 값 추출
5. 텍스트 파일에서 어떤 문자 수정
6. ascii 파일의 어떤 영역에서 어떤 값을 다른 값으로 수정
7. 파일 이름 변경
8. 파일 이름을 일련번호 혹은 날자포맷(yyyymmddHHmm)으로 수정
9. 최대 3개의 ascii 래스터 파일을 이용해서 간단한 대수연산, 조건연산, 조건연산에서 대수연산 포함
